## 🚀 快速開始方式

### 1. 本地測試（推薦）
```bash
cd garbage-main
python3 -m http.server 8000
# 打開 http://localhost:8000
```

### 2. GitHub Pages 部署（免費）
```bash
git init
git add .
git commit -m "Initial commit"
git push -u origin main
# 在 Settings → Pages 中啟用
# 訪問 https://你的用戶名.github.io/garbage
```

### 3. Netlify 部署（最簡單）
1. 訪問 https://netlify.com
2. 連接 GitHub 倉庫
3. 設置 Publish directory = `garbage-main`
4. 自動部署完成！

### 4. Vercel 部署（最快）
```bash
npx vercel
```
